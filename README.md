RentUP — Property Rental PlatformA full-stack web app for searching, listing and booking rental properties. Built with the MERN stack (MongoDB, Express, React, Node.js) with PayPal payments and Cloudinary image uploads.What it does
Tenants can search properties, shortlist them using the Dwell Deck, and book with PayPal
Property owners can create and manage their listings with image uploads
Admins can manage users, listings and bookings from a dedicated panel
Tech Stack
Frontend — React, Redux Toolkit, React Router
Backend — Node.js, Express.js
Database — MongoDB Atlas
Payments — PayPal REST API (sandbox)
Images — Cloudinary
Auth — JWT tokens stored in HttpOnly cookies
